# 🎓 Student Performance Predictor

## 📌 Overview

The **Student Performance Predictor** is a machine learning project that predicts a student's academic performance based on various factors such as study time, attendance, education level, and other personal or academic attributes.

This project helps demonstrate how data can be used to make predictions and support decision-making in education.

---

## 🚀 Features

* Predict student performance using machine learning
* Data preprocessing and cleaning
* Model training and evaluation
* Performance metrics like MAE and R² Score
* Easy-to-understand results

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## 📊 Dataset

The dataset contains information such as:

* Hours_Studies
* Attendances
* Previous_Score
* Final_Score

---

## ⚙️ How It Works

1. Data is collected and cleaned
2. Features are selected and encoded
3. Model is trained using regression techniques
4. Predictions are made on new data
5. Model performance is evaluated

---

## 📈 Model Evaluation

The model was evaluated using:

* **MAE (Mean Absolute Error)** – measures prediction accuracy
* **R² Score** – shows how well the model explains the data

---
